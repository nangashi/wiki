# テスト効果

## 概要

記憶から情報を引き出す検索練習（retrieval practice）は、同じ時間の再学習より記憶定着を大幅に高める。Roediger & Karpicke（2006）の実験では、学習直後は再読群が優位だったが1週間後には想起練習群が大幅に上回った。この現象をテスト効果（Testing Effect）と呼ぶ。

## 詳細

### なぜ起きるか

3つのメカニズムが提唱されている。

**精緻化検索（Elaborative Retrieval）**: 困難な検索タスクは大きな認知的努力を要求し、意味記憶ネットワーク内の関連情報への活性化を広げる。この「検索困難感（desirable difficulty）」を乗り越える瞬間が記憶経路を強化する。

**符号化可変性（Encoding Variability）**: 検索行為は既存の記憶痕跡に新たな検索手がかりを追加する。手がかりが増えるほど、文脈が変わった場面でも思い出せる可能性が高まる。

**神経科学的根拠**: fMRI研究では、検索練習が海馬の前後部に対して「二重作用」を通じて保持を強化することが示されている。再読にはこの神経活動パターンが生じない。

### アクティブリコール

テスト効果を実践に変えた手法がアクティブリコール（Active Recall）である。「情報を見ながら答える」再認を排除し、「情報を見ずに引き出す」想起だけを強制する状況を作ることが本質となる。

| 手法 | 説明 |
|------|------|
| **白紙法** | テキストを閉じ、覚えていることを白紙に全部書き出す |
| **練習問題・過去問** | 教材を読んだ後すぐに問題を解く（解いてから教材に戻る） |
| **ファインマン技法** | その分野を知らない人に教えるつもりで言葉にする |
| **フラッシュカード** | AnkiなどのSRS（間隔反復）ツールで自己テスト |

### 分散学習との組み合わせ

テスト効果は「時間を空けてから思い出す」分散学習と組み合わせることで最大化する。直後の復習より、忘れかけた頃に引き出す方が記憶経路への刺激が強い。エビングハウスの忘却曲線が示す通り、想起のタイミングが記憶強度を左右する。
