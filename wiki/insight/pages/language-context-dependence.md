# 文脈依存度と言語体系

## 概要

低文脈の言語体系は、自然習得の足場である「文脈による意味の補完」そのものを排除するため、使用するだけでは習得されず明示的な訓練を要する。

同じ言語に見えても、その内部には文脈依存度の異なる複数の言語体系（レジスタ：使用場面に応じて語彙・文法の選択が体系的に変わる言語変種）が併存している。高文脈の体系（日常会話の生活言語）は、状況・表情・共有知識が意味を確定してくれることを足場に、使う中で自然に身につく。低文脈の体系（教科書の学習言語、契約書など）は「テキストだけで意味が自己完結する」ことが要求される体系であり、この足場が定義上存在しない。だから一方が流暢でも他方は自動的には習得されない。

## 詳細

### 文脈依存度という軸

レジスタが分岐する軸は複数あるが（Hallidayは場面・関係・伝達様式の3変数で整理した）、習得のされ方を分けるのは文脈依存度である。文脈による補完が使えない体系では、意味の確定をテキスト内部の規則だけで行う必要があり、次の性質がすべて必然的に要求される：

| | 高文脈体系（例：生活言語） | 低文脈体系（例：学習言語） |
|--|--|--|
| 意味の確定 | 状況・表情・共有知識で補完 | テキストが自己完結 |
| 曖昧さ | 許容（あとで修正できる） | 最小化 |
| 主観的表現 | 豊富 | 排除 |
| 条件・例外 | 省略されることが多い | 明示が原則 |
| 指示 | 「これ」「あれ」で足りる | 指示対象を言語化 |
| 習得経路 | 使用の中で自然習得 | 明示的訓練が必要 |

表の最下行（習得経路）は独立の性質ではなく、最上行（意味の確定方法）の帰結である。次節がその機構を説明する。

### なぜ低文脈体系は自然習得されないか

自然習得は文脈補完に依存して動く機構である。会話では状況と相手の反応が意味を確定してくれるため、規則を明示的に知らなくても試行錯誤で言語が使え、誤解は即座に修正される。低文脈体系はまさにこの補助を排除することが定義であるため、日常的な言語使用をいくら積んでもその体系の規則（厳密な係り受け・条件明示・無曖昧性）が顕在化する場面がなく、自然習得の機構が働く余地がない。

この構造は実証的にも支持されている。Cummins（1979）は日常会話力（BICS）と認知的学習言語力（CALP）が別の能力であることを示し、後にこれを「文脈埋め込み（context-embedded）／文脈縮減（context-reduced）」という文脈依存度の軸で定式化した。Schleppegrell（2001）は学校で要求されるレジスタを言語学的に分析し、それが全ての子どもが親しんでいる日常のやり取りの言語とは語彙・文法レベルで異なる体系であること、その発達には「そのレジスタが機能する場面の社会的経験」が必要で日常会話の経験では代替できないことを示した。

### 代表例：生活言語と学習言語

低文脈体系の最たる例が学習言語（教科書・文書の読み書きで使われる言語）である。教科書は読者の状況も表情も知らないままあらゆる読者に同じ意味を伝えなければならず、自己完結制約が最も強く働く。新井紀子はRST（リーディングスキルテスト）のデータで、生活言語が流暢でも学習言語の読解が別問題であることが日本語・日本の学校教育でも成立することを実証した。

### 領域ごとの厳しさの差

自己完結制約の強さは領域ごとに異なるため、低文脈体系はさらに領域固有の変種に分かれる：

- **数学**: 例外がない。「ざっくり言うと」は使わない。定義は完全
- **理科**: 例外を前提に「近似的にこうなる」という記述が標準。「○○の場合を除いて」という条件が頻出

この差異が意識されないまま読むと、理科の教科書を数学の文体として読む誤読（「これは必ず成り立つはずだ」）が生じる。

### 実践への示唆

「低文脈体系は文脈補完の足場がないため自然習得されない」という核心は、双方向の実践を導く：

- **低文脈側へ向かう（教える・学ぶ）**: 学習言語は明示的訓練の対象として扱う。「自分の言葉で」より「テキストの通りに」を先に求める。読解力の停滞は努力不足でなく低文脈体系の未習得を疑う
- **高文脈側へ向かう（伝える）**: 低文脈体系を未習得の相手（小学生など）に届けたければ、文脈補完が働く高文脈の言語（具体・場面・感情）に翻訳する。抽象的な価値語が届かないのは、相手の言語体系に意味確定の足場がないからである
- **領域間の移動**: 数学は例外なし・理科は近似前提という文体の違いを意識して読むと読み誤りが減る
