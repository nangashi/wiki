# 成長マインドセットと固定マインドセット

## 概要

「自分の能力は伸ばせるか、それとも固定しているか」という信念が、挑戦への行動・失敗への反応・長期的成果を決定する。固定マインドセットは自己証明の回路を作り（失敗は能力欠如の証拠 → 挑戦を避ける）、成長マインドセットは学習の回路を作る（失敗は習熟の途中 → 挑戦を続ける）。キャロル・S・ドゥエック（スタンフォード大学）が20年以上の研究で実証。

## 詳細

### 2つのマインドセットの構造

| | 固定マインドセット（Fixed Mindset） | 成長マインドセット（Growth Mindset） |
|---|---|---|
| 能力の捉え方 | 生まれつき決まっており変わらない | 努力・方法・他者の助けで伸ばせる |
| 失敗の意味づけ | 自分の能力の証明（自己の限界） | 習熟の途上にある一時的な状態 |
| 挑戦への態度 | リスクを避ける（失敗 = 自己否定） | 困難を選ぶ（挑戦 = 成長機会） |
| 批判・フィードバック | 脅威として防衛的に受け取る | 有用な情報として取り込む |
| 他者の成功 | 脅威（相対的に自分が劣って見える） | 参考・刺激になる |

### なぜ信念が結果を決めるか

「能力が固定されている」と信じると、あらゆる状況が「能力テスト」になる。失敗 = 能力の欠如を意味するため、挑戦回避・努力の放棄・他責化が合理的選択になる。この回路は自己強化する——難しいことを避けるほど能力は伸びず、伸びないからさらに固定信念が強まる。

「能力が伸ばせる」と信じると、困難は学習機会になる。努力の量が成果に直結すると感じられるため、粘り強さが維持される。ドゥエックの実験では、同じ能力スタート地点でも信念の違いが数年後の学業成果に有意な差を生んだ。

### 「まだ」（Yet）の介入

失敗したとき「できない」と言う代わりに「まだできない」と言い換えるだけで、固定マインドセットから成長マインドセットへの微小な転換が起きる。「まだ」は失敗を最終状態でなく進行中の状態として再定義する。これは核心の「失敗の意味づけ」に直接干渉する介入であり、「失敗 = 能力欠如の証拠（自己証明の回路）」という解釈を「失敗 = 習熟の途中（学習の回路）」へ切り替えることで、挑戦回避から挑戦継続へと行動を変える。

### 偽成長マインドセット（False Growth Mindset）

「努力すれば何でもできる」という過度な一般化はドゥエック自身が警告する誤用。成長マインドセットは「努力が必ず報われる」ではなく「努力・方法・他者からの学びを通じて能力は発達する」というより精密な信念。また、口で「成長マインドセットだ」と言いながら内面が固定マインドセットのまま（＝成果を焦る、失敗を隠す）という状態は介入効果を持たない。

### 適用範囲と限界

スポーツ・学業・ビジネス・対人関係など領域横断で適用できる。ただし成長マインドセット介入の効果規模は小さく（大規模メタ分析）、努力が報われない環境（評価システムの問題）では効果を持たない。信念の変化だけでなく、構造的な機会の存在が前提になる。
