# 学習言語と生活言語

## 概要

学習言語が生活言語と別習得を要するのは、テキストが文脈・表情・状況に頼らず自己完結しなければならないという原理的な制約があるからであり、その制約を満たす読み書き能力は日常会話の自然習得では培われない。

言語には日常会話で自然習得される「生活言語」と、教科書・文書の読み書きで使われる「学習言語」という質的に異なる2種類がある。後者は各教科固有の厳密な表現体系を持ち、生活言語が流暢でも自動的には習得されない。この区別を学校教育が十分に意識してこなかったことが、シン読解力が体系的に教えられてこなかった構造的原因である。

## 詳細

### 生活言語と学習言語の特徴

テキストが自己完結しなければならないという原理的制約から、学習言語には文脈依存度の低さ・曖昧さの最小化・条件の明示が必然的に要求される。生活言語はその逆の性質を持つため、両者は習得経路も使われ方も質的に分岐する。

| | 生活言語 | 学習言語 |
|--|--|--|
| 習得経路 | 日常会話で自然習得 | 明示的指導が必要 |
| 文脈依存度 | 高い（状況・表情で補完） | 低い（テキストが自己完結） |
| 曖昧さ | 許容 | 最小化 |
| 主観的表現 | 豊富 | 排除 |
| 条件・例外の明示 | 省略されることが多い | 明示が原則 |

### 教科固有性

「テキストが自己完結しなければならない」という制約は教科によって適用される厳しさが異なるため、学習言語は教科ごとにさらに異なる表現体系を持つ：

- **数学**: 例外がない。「ざっくり言うと」は使わない。定義は完全
- **理科**: 例外を前提に「近似的にこうなる」という記述が標準。「○○の場合を除いて」という条件が頻出

この差異が意識されないまま読むと、理科の教科書を数学の文体として読む誤読（「これは必ず成り立つはずだ」）が生じる。

### 習得されない理由

学習言語の習得には明示的指導が必要という原理にもかかわらず、学校はその指導を行ってこなかった。生活言語は幼少期から自然に習得されるが、学習言語は学校の教科書に初めて出会う。しかし学校は「自分の言葉で表現する力」を重視するあまり、学習言語そのものを読む訓練を体系的に行ってこなかった。「誰でも真面目に読めば読める」という教育現場の思い込みがその背景にある。

### Cumminsの区別との対応

言語学者Jim Cummins（1979）が提唱したBICS（Basic Interpersonal Communication Skills：日常会話力）とCALP（Cognitive Academic Language Proficiency：認知的学習言語力）の区別と同構造。新井紀子はRST（リーディングスキルテスト）のデータで、この区別が日本語・日本の学校教育においても成立することを実証的に示した。

### 実践への示唆

「学習言語は意識的に教えなければ習得されない」という制約は、読む・教える・教わる場面それぞれで向き先が異なる。

- **教える立場**: 教科書の読み方を「テキストが自己完結している前提で読む」訓練として明示的に扱う。「自分の言葉で」より「テキストの通りに」を先に求める
- **学ぶ立場**: 読解力の停滞は努力不足でなく学習言語未習得に原因があることを疑う（生活言語が流暢でも学習言語は別問題）
- **教科ごとの厳密さの差**: 数学は例外なし・理科は近似前提という文体の違いを意識して読むと読み誤りが減る
