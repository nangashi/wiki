# 提案: [[dokugaku-muscles]] に「5筋」分類の根拠を追加する

## 対象

- `wiki/insight/pages/dokugaku-muscles.md`

## 判定

部分不合格。分類・タクソノミー型だが、5分類の必然性が十分に説明されていない。

## 根拠

概要は「独学を可能にする5種の認知能力」と述べているが、なぜこの5つで切るのか、他の分類では何が説明できなくなるのかが明確でない。`essence-review-criteria.md` の C6「分類の根拠」に照らすと不足している。

## 改善案

詳細冒頭に「5筋は独学プロセスの詰まり箇所に対応している」という分類根拠を追加する。

例:

> 5つの独学筋は能力名の一覧ではなく、独学が止まる5つの詰まり箇所に対応する。疑問筋は問いが立たない詰まり、差分筋は違いを抽出できない詰まり、関連筋は既有知識へ接続できない詰まり、検証筋は理解を試せない詰まり、継続筋は探索を維持できない詰まりを扱う。この分類軸により、「学習量不足」ではなく「どの認知操作が詰まっているか」を診断できる。

実際の5筋名が本文と異なる場合は、本文の分類に合わせて書き換える。
